                          使用指南
================================================================================

快速开始:

1. 验证安装
   python verify_migration_tool.py

2. 测试连接
   python manage_migrations.py test

3. 应用迁移
   python manage_migrations.py upgrade

4. 运行测试
   pytest test_migrations.py -v

常用命令:

  创建迁移: python manage_migrations.py create "description"
  应用迁移: python manage_migrations.py upgrade
  回滚迁移: python manage_migrations.py downgrade
  查看状态: python manage_migrations.py check
  查看历史: python manage_migrations.py history
