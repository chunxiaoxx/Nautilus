### 告警规则 (18条)

**Critical (5条)**:
- API错误率 > 5%
- 数据库连接失败
- 区块链连接断开
- 数据库错误率过高
- 异常API调用

**Warning (11条)**:
- API响应时间 > 2秒
- API请求量异常下降
- 数据库查询慢
- 区块链交易失败率 > 10%
- Gas费用 > 100 Gwei
- 待处理交易过多
- 任务失败率 > 20%
- 任务队列积压
- CPU使用率 > 80%
- 内存使用率过高
- 登录失败频繁
- 权限拒绝频繁

**Info (2条)**:
- WebSocket连接数低
- Redis连接失败

## 📝 使用示例

### 启动监控系统

```bash
# 一键启动
bash start_monitoring.sh

# 查看服务状态
docker-compose -f docker-compose.monitoring.yml ps

# 查看日志
docker-compose -f docker-compose.monitoring.yml logs -f prometheus
```

### 查看监控指标

```bash
# 查看所有指标
curl http://localhost:8000/metrics

# 查看安全事件指标
curl http://localhost:8000/metrics | grep nautilus_security

# 查看健康状态
curl http://localhost:8000/health
```

### 分析安全日志

```bash
# 分析最近24小时日志
python monitoring/analyze_security_logs.py --log-dir logs --hours 24

# 生成报告并打印
python monitoring/analyze_security_logs.py --print

# 导出JSON格式
python monitoring/analyze_security_logs.py --json security_events.json
```
