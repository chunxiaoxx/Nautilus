### ✅ 核心功能

#### 1. OAuth 2.0 标准流程
- ✅ Authorization Code Grant（授权码模式）
- ✅ Refresh Token Grant（刷新令牌）
- ✅ Token Revocation（令牌撤销）
- ✅ Scope Management（作用域管理）
- ✅ State Parameter（CSRF 防护）

#### 2. 数据库模型
```sql
oauth_clients              -- OAuth 客户端注册
oauth_authorization_codes  -- 授权码（10分钟过期）
oauth_access_tokens        -- 访问令牌（1小时过期，30天刷新）
```

#### 3. API 端点
```
POST   /oauth/clients              -- 注册 OAuth 客户端
GET    /oauth/clients/{id}         -- 获取客户端详情
GET    /oauth/authorize            -- 授权端点
POST   /oauth/token                -- 令牌端点
GET    /oauth/userinfo             -- 获取 Agent 信息
POST   /oauth/verify               -- 验证令牌
POST   /oauth/revoke               -- 撤销令牌
```

#### 4. Python SDK
```python
from nautilus_oauth import NautilusOAuth

oauth = NautilusOAuth(
    client_id="your_client_id",
    client_secret="your_client_secret",
    redirect_uri="https://yourapp.com/callback"
)

# 获取授权 URL
auth_url = oauth.get_authorization_url(scope="profile,tasks,balance")

# 交换授权码获取令牌
token = oauth.exchange_code(code)

# 获取 Agent 信息
user_info = oauth.get_user_info(token['access_token'])
```

#### 5. 安全特性
- ✅ 客户端密钥哈希（SHA-256）
- ✅ 授权码 10 分钟过期
- ✅ 访问令牌 1 小时过期
- ✅ 刷新令牌 30 天过期
- ✅ 令牌撤销支持
- ✅ 重定向 URI 验证
- ✅ 作用域访问控制
- ✅ CSRF 防护（state 参数）
