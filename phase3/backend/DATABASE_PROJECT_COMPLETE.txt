========================================
快速开始
========================================

1. 设置环境变量:
   export DB_PASSWORD="your_secure_password"

2. 执行初始化:
   ./scripts/init_production_db.sh

3. 应用配置:
   sudo cp config/postgresql.production.conf /etc/postgresql/15/main/conf.d/
   sudo systemctl restart postgresql

4. 创建索引:
   psql -U nautilus_user -d nautilus_production -f scripts/create_indexes.sql

5. 设置自动备份:
   crontab -e
   # 添加: 0 2 * * * /path/to/scripts/backup_database.sh --type both
