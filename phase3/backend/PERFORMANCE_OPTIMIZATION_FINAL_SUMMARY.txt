四、部署步骤
================================================================================

1. 应用数据库索引
   $ python add_performance_indexes.py

2. 配置环境变量 (.env)
   DATABASE_POOL_SIZE=20
   DATABASE_MAX_OVERFLOW=40
   DATABASE_POOL_RECYCLE=3600
   SLOW_QUERY_THRESHOLD=0.3
   CACHE_DEFAULT_TTL=300

3. 运行验证脚本
   $ bash verify_performance_optimization.sh

4. 运行性能测试
   $ pytest tests/test_performance.py -v

5. 运行基准测试
   $ python benchmark_performance.py

6. 启动应用并监控
   $ python main.py
   访问监控端点:
   - http://localhost:8000/cache/stats
   - http://localhost:8000/performance/stats
   - http://localhost:8000/database/pool

五、监控端点

3. GET  /performance/stats    - 性能统计 (请求计数、响应时间)
4. POST /performance/reset    - 重置性能统计
5. GET  /database/pool        - 数据库连接池状态
