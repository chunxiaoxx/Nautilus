# Nautilus Pitch Deck - PPT制作指南

## 🎨 设计规范

### 配色方案

**主色调**:
- 主蓝色: #0066FF (科技感)
- 深蓝色: #003366 (专业感)
- 金色: #FFB800 (价值感)
- 白色: #FFFFFF (背景)
- 深灰: #333333 (文字)
**使用规则**:
- 标题: 深蓝色
- 正文: 深灰色
- 强调: 主蓝色或金色
- 背景: 白色或浅灰渐变

### 字体规范

**中文**:
- 标题: 思源黑体 Bold, 36-48pt
- 副标题: 思源黑体 Medium, 24-32pt
- 正文: 思源黑体 Regular, 18-24pt

**英文**:
- 标题: Helvetica Neue Bold, 36-48pt
- 副标题: Helvetica Neue Medium, 24-32pt
- 正文: Helvetica Neue Regular, 18-24pt

### 布局规范

**页面尺寸**: 16:9 (1920x1080px)
**边距**: 上下左右各80px
**标题区**: 顶部150px
**内容区**: 中间700px
**底部**: Logo + 页码, 50px

## 🎬 动画建议

### 全局动画
- 页面切换: 淡入淡出
- 标题: 从左滑入
- 内容: 依次出现
- 图表: 动态绘制

### 特殊效果
- Logo: 放大+旋转
- 数字: 滚动计数
- 曲线: 动态绘制
- 饼图: 扇形展开

### 时间控制
- 每个动画: 0.5-1秒
- 页面停留: 1-2分钟
- 总时长: 15-20分钟
