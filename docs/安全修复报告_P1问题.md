## P1问题清单

### 1. 环境变量验证不足 (5个问题)

#### 1.1 OAuth配置缺少验证
**文件**: `api/auth.py`
**行号**: 43-46
**问题**:
```python
GITHUB_CLIENT_ID = os.getenv("GITHUB_CLIENT_ID")
GITHUB_CLIENT_SECRET = os.getenv("GITHUB_CLIENT_SECRET")
GITHUB_REDIRECT_URI = os.getenv("GITHUB_REDIRECT_URI")
FRONTEND_URL = os.getenv("FRONTEND_URL", "http://localhost:3000")
```

**风险**: OAuth配置为空时，系统仍然运行，但OAuth功能会失败，错误信息可能泄露配置问题。

**修复方案**:
- 添加启动时验证
- 如果OAuth未配置，禁用相关端点
- 记录警告日志

#### 1.2 Google OAuth配置未验证
**文件**: `api/auth.py`
**行号**: 48-50
**问题**: 同上

#### 1.3 区块链配置缺少验证
**文件**: `blockchain/web3_config.py`
**问题**: INFURA_PROJECT_ID可能为空

#### 1.4 Redis配置缺少验证
**文件**: 多个文件使用Redis
**问题**: Redis连接失败时，错误处理不一致

#### 1.5 LLM API密钥未验证
**文件**: `.env`
**问题**: OPENAI_API_KEY和ANTHROPIC_API_KEY使用占位符

### 2. 错误信息泄露 (4个问题)

#### 2.1 数据库错误泄露
**文件**: 多个API端点
**问题**: 数据库异常直接返回给客户端

**示例**:
```python
except Exception as e:
    raise HTTPException(status_code=500, detail=str(e))
```

**风险**: 泄露数据库结构、表名、字段名

**修复方案**:
```python
except Exception as e:
    logger.error(f"Database error: {e}")
    raise HTTPException(status_code=500, detail="Internal server error")
```

#### 2.2 区块链错误泄露
**文件**: `api/tasks.py`, `api/agents.py`
**问题**: 区块链交易失败时，错误详情返回给客户端

#### 2.3 文件路径泄露
**文件**: `blockchain/key_manager.py`
**问题**: 日志中包含完整文件路径

#### 2.4 API Key格式泄露
**文件**: `api/agent_tasks.py`
**问题**: 错误消息"Invalid API Key format"可能帮助攻击者

---

### 3. 速率限制缺失 (3个问题)

#### 3.1 管理端点缺少速率限制
**文件**: `main.py`
**端点**: `/cache/clear`, `/performance/reset`
**问题**: 管理端点没有速率限制，可能被滥用

**修复方案**:
```python
@app.post("/cache/clear")
@limiter.limit("5/hour")  # 添加严格限制
async def clear_cache(request: Request, admin: User = Depends(get_current_admin_user)):
    ...
```

#### 3.2 健康检查端点限制过宽
**文件**: `main.py`
**端点**: `/health`
**当前**: 50/minute
**建议**: 30/minute

#### 3.3 CSRF token端点无限制
**文件**: `main.py`
**端点**: `/csrf-token`
**问题**: 可能被用于DDoS攻击

---

### 4. 输入验证不足 (3个问题)

#### 4.1 任务描述长度未限制
**文件**: `api/tasks.py`
**模型**: `TaskCreate`
**问题**: description字段没有最大长度限制

**修复方案**:
```python
class TaskCreate(BaseModel):
    description: str = Field(..., max_length=5000)
    input_data: Optional[str] = Field(None, max_length=10000)
    expected_output: Optional[str] = Field(None, max_length=10000)
```

#### 4.2 Agent名称未验证
**文件**: `api/agents.py`
**问题**: Agent名称可能包含特殊字符或过长

#### 4.3 钱包地址格式验证不一致
**文件**: 多个文件
**问题**: 有些地方验证，有些地方不验证

---

### 5. 日志敏感数据 (3个问题)

#### 5.1 API Key记录到日志
**文件**: `api/agents.py:line 某行`
**问题**:
```python
logger.info(f"Generated API key for agent: {agent.agent_id}")
```

**风险**: 虽然没有记录完整API Key，但agent_id可能敏感

**修复方案**:
```python
logger.info(f"Generated API key for agent: {agent.agent_id[:8]}...")
```

#### 5.2 区块链私钥路径泄露
**文件**: `blockchain/key_manager.py`
**问题**: 日志中包含密钥文件路径

#### 5.3 用户信息记录过详细
**文件**: 多个认证端点
**问题**: 登录失败时记录完整用户信息
