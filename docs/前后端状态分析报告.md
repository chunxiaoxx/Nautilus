## 🔗 前后端集成状态

### 🟡 配置不一致问题

**前端API配置**:
- 代码中: `http://localhost:3000/api` (api.ts)
- 环境变量: `https://api.nautilus.social` (.env)
- 实际后端: `http://localhost:8001`

**问题分析**:
1. 前端代码默认指向 `localhost:3000/api`
2. 后端运行在 `localhost:8001`
3. 缺少Vite代理配置将 `/api` 转发到 `8001`
4. 生产环境配置指向 `api.nautilus.social`

### 🔧 需要修复的配置

**1. Vite代理配置缺失**
```typescript
// vite.config.ts 需要添加
server: {
  port: 3000,
  proxy: {
    '/api': {
      target: 'http://localhost:8001',
      changeOrigin: true,
      rewrite: (path) => path.replace(/^\/api/, '')
    }
  }
}
```

**2. 环境变量配置**
```env
# .env.development (需要创建)
VITE_API_BASE_URL=http://localhost:8001
VITE_WS_URL=ws://localhost:8001/ws

# .env.production (需要更新)
VITE_API_BASE_URL=https://api.nautilus.social
VITE_WS_URL=wss://api.nautilus.social/ws
```
