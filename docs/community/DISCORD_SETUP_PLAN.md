# Nautilus Discord 社区搭建计划

**创建日期**: 2026-03-19
**目标**: 打造活跃的开发者/AI爱好者社区

---

## 一、社区定位

### 核心价值
- **学习**: Nautilus 使用教程、技巧分享
- **交流**: 开发者讨论、问题解答
- **展示**: 作品分享、案例分析
- **协作**: 共同开发、开源贡献

### 目标用户
- AI/ML 开发者
- 产品经理
- 技术爱好者
- 学生/研究者

---

## 三、Bot 配置

### 必备 Bot

| Bot | 功能 | 优先级 |
|-----|------|--------|
| Carl-bot | 欢迎消息、Moderation | P0 |
| MEE6 | 等级、奖励 | P1 |

### 自动消息

**欢迎消息**:
```
👋 欢迎来到 Nautilus 社区！

在这里你会找到：
✅ 184个专业AI Agent
✅ 完整的开发文档
✅ 活跃的开发者社区

请先看看 #rules，然后自我介绍一下吧！

🚀 github.com/chunxiaoxx/nautilus-core
```

**规则**:
```
📜 社区规则

1. 保持友好 - 尊重每一个人
2. 提问前先搜索 - 避免重复问题
3. 分享代码要格式化 - 用代码块
4. 不要刷屏 - 给别人机会
5. 欢迎新人 - 他们是社区的未来
```

## 四、运营策略

### 欢迎仪式
- 新成员自动欢迎
- 要求自我介绍
- 推荐关注 #resources

|------|------|------|
