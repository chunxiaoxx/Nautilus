## 📋 实施准备（两个方案）

### 方案A: nautilus.social

#### 1. 域名注册
```bash
# 推荐注册商
- Namecheap: https://www.namecheap.com
- GoDaddy: https://www.godaddy.com
- Cloudflare: https://www.cloudflare.com

# 预估费用: $50/年
```

#### 2. DNS配置
```bash
# A记录配置
类型    主机记录    记录值              TTL
A       @          43.160.239.61       600
A       www        43.160.239.61       600
A       api        43.160.239.61       600
```

#### 3. SSL证书获取
```bash
# 使用Let's Encrypt
sudo certbot certonly --nginx -d nautilus.social -d www.nautilus.social -d api.nautilus.social
```

#### 4. Nginx配置
```bash
# 更新配置文件
sudo nano /etc/nginx/sites-available/nautilus

# 替换所有域名引用
:%s/nautilus\.ai/nautilus.social/g

# 重启Nginx
sudo nginx -t
sudo systemctl reload nginx
```
