## 💡 部署建议

### 禅心AI APP部署方案

#### 方案A: Docker部署（推荐）
```bash
# 1. 创建项目目录
sudo mkdir -p /opt/zenxin-ai
cd /opt/zenxin-ai

# 2. 克隆代码
git clone <your-repo> .

# 3. 使用Docker Compose
docker compose up -d

# 4. 配置Nginx反向代理
```

**优点**:
- 隔离环境
- 易于管理
- 可以和Nautilus共存

#### 方案B: 直接部署
```bash
# 1. 安装依赖
npm install

# 2. 构建
npm run build

# 3. 使用PM2运行
pm2 start npm --name "zenxin-ai" -- start
```
