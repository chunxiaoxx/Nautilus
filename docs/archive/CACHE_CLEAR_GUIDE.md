## 🔄 彻底清除缓存的方法

### 方法1: 硬刷新（最简单）

1. 打开 https://www.nautilus.social/login
2. 按 `Ctrl + Shift + R`（Windows）或 `Cmd + Shift + R`（Mac）
3. 或者按 `Ctrl + F5`
4. 多按几次确保刷新

### 方法2: 开发者工具强制刷新

1. 按 `F12` 打开开发者工具
2. **右键点击**浏览器刷新按钮
3. 选择"清空缓存并硬性重新加载"
4. 或者在Network标签勾选"Disable cache"后刷新

### 方法3: 手动清除缓存

#### Chrome/Edge:
1. 按 `Ctrl + Shift + Delete`
2. 选择"缓存的图片和文件"
3. 时间范围选"全部时间"
4. 点击"清除数据"
5. **完全关闭浏览器**（所有窗口）
6. 重新打开浏览器

#### Firefox:
1. 按 `Ctrl + Shift + Delete`
2. 选择"缓存"
3. 时间范围选"全部"
4. 点击"立即清除"
5. **完全关闭浏览器**
6. 重新打开浏览器

### 方法4: 隐私/无痕模式

1. 按 `Ctrl + Shift + N`（Chrome）或 `Ctrl + Shift + P`（Firefox）
2. 在无痕窗口访问 https://www.nautilus.social/login
3. 尝试登录
