## 💡 技术亮点

### ACK/NACK消息确认机制

#### ACK消息（确认）
```python
ack_msg = create_ack_message(
    from_agent='agent-a',
    to_agent='agent-b',
    ack_message_id='msg-123',
    status='received'  # received/processing/completed
)
```

**用途**:
- 确认消息已接收
- 报告处理状态
- 追踪消息流转

#### NACK消息（拒绝）
```python
nack_msg = create_nack_message(
    from_agent='agent-a',
    to_agent='agent-b',
    nack_message_id='msg-456',
    reason='Invalid format',
    error_code='ERR_001',
    retry_possible=True,
    retry_after=60
)
```

**用途**:
- 拒绝无效消息
- 提供错误信息
- 支持重试机制
