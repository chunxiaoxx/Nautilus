## 一、宇树科技（Unitree Robotics）

### 1. 公司概况

| 项目 | 信息 |
|------|------|
| **成立时间** | 2016年5月 |
| **总部** | 浙江杭州 |
| **创始人/CEO** | 王兴兴（Wang Xingxing） |
| **员工规模** | 1,000+ 人（2025年数据） |
| **估值** | 约17亿美元（独角兽，Series C） |
| **融资总额** | 约1.55亿美元 |
| **主要投资方** | 深圳资本集团、腾讯、顺为资本 |
| **年营收** | 约1.4亿美元（2025年） |

宇树科技是全球四足机器人行业先驱，2016年从一人公司起步，早期以消费级四足机器狗（Unitree A1、Go2）闻名，曾两度登上央视春晚（2021牛年春晚、2022北京冬奥会开幕式）。2023年起切入人形机器人赛道，成为国内最早同时具备四足+人形双产品线的机器人公司。

### 2. 核心产品

**人形机器人：**
- **Unitree H1**（"火鸡"）
  - 身高约180cm，体重约47kg
  - 移动速度最高3.3m/s（全球最快双足人形之一）
  - 峰值扭矩密度 189 N·m/kg
  - 定价：$90,000（研究/行业平台）
  - 定位：科研+工业应用平台

- **Unitree G1**
  - 身高约127cm（4英尺2英寸）
  - 定位：轻工业/服务/教育场景
  - 定价：$16,000（基础配置）—全球最便宜的人形机器人之一
  - 支持 Unitree Explore APP 远程连接控制

**四足机器人：**
- **Go2系列**：消费级/教育级机器狗，定价亲民
- **B2-W**：工业级四足，可爬楼梯、越障，面向电力巡检等场景

### 3. 技术特点

- **自研核心零部件**：自研电机、减速器、驱动器、编码器等关键关节模组
- **行业首创4D LiDAR L1**：宇树自研的4D激光雷达，与机器人深度集成
- **成本控制能力**：依托国内供应链，将人形机器人价格打到$16K–$90K区间
- **运动控制积累**：四足机器人多年的运动控制算法积累，可迁移至人形
- **开发者生态**：有配套文档中心（support.unitree.com）和APP

### 4. 与Nautilus合作可能性评估

| 维度 | 评分 | 分析 |
|------|------|------|
| 硬件互补性 | ★★★★☆ | 四足+人形双线，Nautilus若专注AI/软件层，可形成分工 |
| 价格友好度 | ★★★★★ | G1仅$16K，降低Nautilus客户或测试场景的硬件门槛 |
| 开发者生态 | ★★★★☆ | 开放文档、SDK，易于集成 |
| 具身AI深度 | ★★★☆☆ | 偏运动控制+硬件，具身AI算法层非核心优势 |
| 合作切入角度 | — | 提供低成本机器人硬件平台；Nautilus输出AI层能力 |

**建议合作形式**：Nautilus与宇树在G1/H1上预装集成具身AI模型，联合打造"AI+硬件"开发套件，面向科研机构和企业客户。

## 二、智元机器人（Agibot / 智元创新）

### 1. 公司概况

| 项目 | 信息 |
|------|------|
| **成立时间** | 2023年2月 |
| **总部** | 上海浦东（漕河泾康桥商务绿洲） |
| **战略定位** | 全球领先的通用具身智能机器人公司 |
| **使命** | 以智能机器创造无限生产力 |
| **愿景** | 智能机器人全球领军企业，开创通用机器人生态 |
| **融资历程** | 2024年3月完成A++++轮（超10亿人民币，M31资本、红杉中国领投）；2025年3月获腾讯领投新一轮（数亿人民币） |
| **累计产量** | 2025年1月通用机器人累计下线1,000台 |

智元是目前国内融资最猛、增长最快的人形机器人初创企业之一。创始团队具有华为等头部科技公司背景，公司文化强调"追求卓越、实干高效、产品为王"。

### 2. 核心产品线

**人形机器人（远征系列）：**
- **远征A2旗舰版（Yuanzheng A2 Ultra）**
  - 全尺寸通用人形机器人，旗舰款
  - 搭载自研灵巧手（OmniHand）
  - VR遥操作套装支持
  
- **远征A2青春版（A2 Lite）**
  - 高性价比全尺寸人形
  - 面向不同场景需求

- **远征A2-W（Yuanzheng A2-W）**
  - 面向柔性制造场景的专用人形

**开源机器人（灵犀系列）：**
- **灵犀X1（X1）**
  - 全栈开源机器人，面向开发者社区
  - AimRT通信框架支持（类似ROS2的机器人通信中间件）
  - 智元灵犀X1已开源文档和部分代码

- **灵犀X2（X2）**
  - 全智能灵动机器人，2025年3月发布
  - 相比X1全面升级

**通用具身智能机器人（精灵系列）：**
- **精灵G2、G1**
  - 通用具身智能机器人
  - 强调环境感知+任务理解+物理执行能力

**四足机器人（酷拓/D1系列）：**
- **D1 Ultra、D1 Pro/Edu**
  - 全能四足智能机器人

**商用清洁机器人：**
- **智元绝尘C5**
  - 首款商用清洁机器人

**配件生态：**
- **OmniHand 2025（灵动款）**：小身形机械手，兼容性强
- **OmniHand Pro 2025（专业款）**：增强感知版
- **VR遥操作套装**：仅适配远征A2

### 3. 技术特点

- **全栈自研**：机器人本体+智能算法+开放平台，三位一体
- **具身基座大模型**：2025年3月发布"智元启元大模型"（具身基座模型）
- **一站式开发平台 Genie Studio**（2025年4月发布）：面向具身智能开发者的完整工具链
- **大规模数据采集能力**：自建3000+平方米数据采集工厂，支持具身AI训练
- **AgiBot World 数据集**：开源大规模具身智能数据集
- **AimRT通信框架**：类ROS2但更现代的机器人通信框架，开源
- **软硬件一体化**：自研灵巧手（OmniHand）+ VR遥操作套装

## 三、傅利叶智能（Fourier Intelligence）

### 1. 公司概况

| 项目 | 信息 |
|------|------|
| **成立时间** | 约2015年（从康复机器人切入） |
| **总部** | 上海 |
| **战略定位** | "Made for AI" — 专注AI驱动的人形机器人 |
| **使命** | 利用全栈机器人技术丰富人类生活 |
| **融资** | 2025年1月完成近8亿元E轮融资 |
| **主要投资方** | 国新投资、浦东创投、张江高科、望源资本、Prosperity7 Ventures（沙特阿美）、君山资本 |
| **产品线** | 康复机器人（RehabHub）+ 人形机器人（GRx系列） |

傅利叶从康复机器人起家，技术积累深厚，2023年起正式切入人形机器人赛道。其独特优势在于FSA（傅里叶智能关节）自研关节模组，已进入第六代。

### 2. 核心产品

**GRx 人形机器人系列：**

- **GR-1**（首发款，定位：首款量产人形机器人）
  - 身高：165cm
  - 体重：55kg
  - 关节数：44
  - 最大行走速度：5 km/h
  - 峰值扭矩：230 N·m
  - 传感器：1个环形麦克风 + 1个RealSense深度相机 + 6RGB相机纯视觉感知方案（BEV+Transformer+Occupancy Network）
  - 定价：面向研究/行业客户

- **GR-2**（新一代里程碑）
  - 身高：175cm
  - 体重：63kg
  - 关节数：53
  - 峰值扭矩：380 N·m（配备FSA 2.0）
  - 电池续航：2小时
  - **12-DoF灵巧手**（双倍于GR-1的灵巧性）
  - 6个阵列式触觉传感器（可感知力、识别物体形状和材质）
  - 双编码器系统，控制精度翻倍
  - 支持NVIDIA Isaac Lab、ROS、Mujoco仿真框架
  - 支持傅里叶工具包（Fourier Toolkit）SDK

- **GR-3系列**（2025年新品，定位：情感陪伴+服务）
  - 身高：165cm
  - 体重：71kg
  - 关节数：55
  - 电池续航：3小时
  - 双可热插拔电池（全天候运行）
  - **全身遥操作**（通过高精度动作捕捉+低延迟传输实现）
  - 面向：公共服务、儿童陪伴、老年人陪伴、研究、康复训练
  - 强调情感连接，而非纯功能工具
  - **GR-3C**：计划扩展至辅助医疗/高危操作/康复训练

### 3. 技术特点

- **FSA（傅里叶智能关节）**：将电机、驱动器、减速器、编码器四合一，高度集成化，显著减小体积同时提升可靠性
  - FSA 2.0版本峰值扭矩超380 N·m
- **纯视觉感知方案**：BEV + Transformer + Occupancy Network融合，6个RGB相机实时生成3D占据栅格
- **触觉感知**：灵巧手配备阵列式触觉传感器，支持力反馈和材质识别
- **双热插拔电池**：GR-3支持全天候不间断运行
- **仿真支持完善**：原生支持Isaac Lab、ROS、Mujoco，降低AI训练门槛
- **模块化设计**：传感器和计算单元模块化集成，便于开发维护

### 4. 与Nautilus合作可能性评估

| 维度 | 评分 | 分析 |
|------|------|------|
| 硬件品质 | ★★★★★ | FSA关节自研，峰值扭矩参数领先，工业级可靠性 |
| 具身AI契合度 | ★★★★☆ | "Made for AI"战略定位明确，GR系列专为AI训练设计 |
| 开发者生态 | ★★★★☆ | Fourier Toolkit SDK + Isaac Lab原生支持 |
| 触觉感知 | ★★★★★ | 阵列式触觉传感器，具身操作任务的关键感知层 |
| 情感交互 | ★★★☆☆ | GR-3强调情感陪伴，区别于工业定位 |
| 合作切入角度 | — | 傅里叶硬件平台+Nautilus具身AI模型=完整人形机器人方案 |

**建议合作形式**：Nautilus具身AI模型在傅里叶GR系列上进行预训练和部署；联合开发面向医疗/服务场景的专用具身AI技能包。

## 七、与Nautilus合作可能性综合评估

### 最优先合作候选

**🥇 智元机器人（Agibot）**
- 原因：战略定位与Nautilus高度重叠（具身AI为核心），拥有自研大模型+Genie Studio生态+开源数据集，合作界面清晰，且腾讯已投资背景，资本层面可对接。具身AI技术深度最强。

**🥈 傅利叶智能（Fourier）**
- 原因：FSA关节自研+触觉感知是硬件差异化优势；"Made for AI"战略明确欢迎AI层合作；具身AI训练配套完善（Isaac Lab/Nijoco仿真支持）。面向研究机构客户的渠道可与Nautilus共享。

**🥉 宇树科技（Unitree）**
- 原因：G1/H1价格友好，是Nautilus算法层的最佳硬件载体之一；开发者生态完善；四足+人形双线可覆盖多种客户需求场景。偏硬件层，AI算法非核心，合作以Nautilus输入AI为主。

### 需持续关注

**自变量机器人（X Square）**：开源基础模型战略+深圳+高融资，2025–2026年预计加速，建议列为中长期战略跟踪对象。

**追觅科技（Dreame）**：家用具身AI场景特殊，数据价值高，但人形机器人产品化仍需时间，适合早期联合研发。

## 八、风险提示

1. **行业同质化风险**：国内人形机器人公司快速涌现，产品差异化和商业化路径尚不明朗
2. **具身AI技术瓶颈**：泛化能力、数据瓶颈、长尾任务等问题仍是行业共性挑战
3. **资本依赖风险**：多数公司仍处于亏损状态，依赖持续融资维系运营
4. **政策/地缘风险**：机器人核心零部件（芯片、关节）存在供应不确定性
