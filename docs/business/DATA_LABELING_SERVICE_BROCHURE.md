## 质量保障

### Raid 共识机制

```
数据项 → Agent A 标注 → Agent B 标注 → Agent C 标注 → 投票共识 → 最终标签
```

- **3 个 Agent 独立标注**每条数据
- **多数投票**确定最终标签（可配置阈值 67%-100%）
- **置信度评分**标注每条标注结果
- **低共识项**自动标记为需人工审核

### 质量指标（实测）

| 指标 | 数值 |
|------|------|
| 共识率 | 100%（5 项测试数据） |
| 平均置信度 | 91.6% |
| 需人工审核比例 | 0% |
| 处理速度 | 5 条 / 18.4 秒 |

## 服务模式

### 模式一：API 接入（推荐）

```
POST /api/labeling/jobs
{
  "name": "车辆检测标注",
  "labeling_type": "object_detection",
  "items": ["场景1描述", "场景2描述", ...],
  "num_agents": 3,
  "consensus_threshold": 0.67
}
```

- 实时返回标注结果
- 支持 CSV 批量上传
- 自动生成质量报告

### 模式二：批量文件处理

- 客户发送 CSV/JSON 文件
- 我们处理后返回标注结果 + 质量报告
- 支持 10,000+ 条批量处理

### 模式三：定制化标注

- 自定义标签体系
- 自定义标注指南
- 专项质量优化

