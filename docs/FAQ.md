# Nautilus FAQ

## 常见问题

### 什么是 Nautilus？

Nautilus 是全球首个基于顶级学术论文的开源 AI Agent 生态系统。它让 AI 从"工具"进化为"生命"，通过协作网络、价值引擎和进化系统，让 AI Agent 能够自主协作、学习和创造价值。

---

### Nautilus 和 Perplexity 有什么区别？

Perplexity 是"AI 查询机"——帮你搜资料、告诉你答案。

Nautilus 是"AI 电脑"——帮你完成任务、直接给出结果。

就像当年：查询机 → 电脑

---

### 需要编程基础吗？

**不需要。**

- Web 界面：完全可视化操作
- 自然语言：用日常语言下达任务
- 有 API：开发者可以深度定制

---

### 支持哪些模型？

Nautilus 支持主流 LLM：

- OpenAI (GPT-4, GPT-4o)
- Anthropic (Claude)
- Google (Gemini)
- MiniMax
- 以及其他兼容 OpenAI API 的模型

---

### 是免费的吗？

**核心功能免费：**

- 开源版本：完全免费
- 基本功能：无限使用

**付费功能：**

- 云端托管服务 (即将上线)
- 企业版 (即将上线)

---

### 如何安装？

**Docker (推荐):**
```bash
docker run -d -p 18789:18789 openclaw/nautilus
```

**npm:**
```bash
npm install -g openclaw
```

---

### 有多少个 Agent？

**184 个**专业 Agent，涵盖：

- 开发 (前端、后端、全栈)
- 设计 (UI、Logo、PPT)
- 营销 (SEO、内容、社媒)
- 销售 (CRM、客服)
- 运营 (数据分析、增长)
- 等等...

---

### Agent 可以协作吗？

**可以。**

Nautilus 的核心特性就是多 Agent 协作：

- 任务自动分解
- Agent 间通信
- 资源共享
- 结果汇总

---

### 如何评估 Agent 价值？

Nautilus 使用 **Epiplexity** 指标：

- **知识创造**：输出了多少有价值的信息
- **学习能力**：从任务中学到了什么
- **协作价值**：对团队的贡献

---

### 和其他 AI Agent 有什么不同？

| 特性 | 其他 AI Agent | Nautilus |
|------|---------------|----------|
| 协作 | 无 | 有 |
| 学习 | 无 | 有 |
| 价值评估 | 无 | Epiplexity |
| 定制 | 有限 | 无限 |
| 开源 | 部分 | 完全 |
